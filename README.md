#internal nedir?

Internal olarak tanımlanan bir değer; aynı program içerisinden erişilebilir, fakat farklı bir program içerisinden erişilemez durumdadır. Program içerisinde herhangi bir kısıtlaması yoktur.
Javada kullanılmaz.

#extend vs implement 
Extend her zaman implements anahtar sözcüğünden önce gelmektedir.
Java derleyicisi bir sınıfı byte koduna derlerken öncelikle bir üst sınıfa bakmalıdır. Çünkü sınıfların altında yatan uygulama ilgili yöntemleri ve alanları tutan üst sınıfın byte kodunu göstermektedir.
